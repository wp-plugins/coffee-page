### Coffee Page                             

Responsive and modern error 404 page.

### The Features

* Just activate the plugin to display error 404 page.
* Responsive design.
* Modern design using CSS3.
* Search form.
* RTL Support.
* Settings panel to translate the texts easily.

== Installation ==

### Installation

1. Upload 'coffee-page' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now plugin is ready, but if you want to translate texts, go to your wp-admin > Coffee Page.
4. Enjoy.
